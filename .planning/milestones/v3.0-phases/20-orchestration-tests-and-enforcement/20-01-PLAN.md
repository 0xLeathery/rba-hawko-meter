### Helper: `_make_z_df(n_rows=15, base_date="2025-01-01")`

Builds a minimal DataFrame with `date`, `value`, `z_score`, `window_size` columns — suitable for use as `z_df` and `latest_row` in `build_gauge_entry` tests.

```python
def _make_z_df(n_rows=15, base_date="2025-01-01"):
    """Create a minimal z-score DataFrame for testing."""
    import pandas as pd
    dates = pd.date_range(base_date, periods=n_rows, freq="QS")
    values = [2.0 + i * 0.1 for i in range(n_rows)]
    z_scores = [0.5 + i * 0.05 for i in range(n_rows)]
    return pd.DataFrame({
        "date": dates,
        "value": values,
        "z_score": z_scores,
        "window_size": [20] * n_rows,
    })
```
